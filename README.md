## 🚀 Getting Started

### Run Locally

1. Clone the repository
   ```bash
   git clone https://github.com/rahulplots/your-repo-name.git
   ```
2. Navigate into the project folder
   ```bash
   cd your-repo-name
   ```
3. Open `index.html` directly in your browser, or use a live server extension (e.g., VS Code's "Live Server") for the best experience.

### Deploy on GitHub Pages

1. Push your code to a GitHub repository
2. Go to **Settings → Pages**
3. Under **Source**, select your branch (usually `main`) and root folder
4. Save — your site will be live at:
   ```
   https://rahulplots.github.io/your-repo-name/
   ```
